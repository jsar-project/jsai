# JSAI

JSAI is a JavaScript-based AI framework designed for the Spatial Web. It empowers developers to build intelligent, interactive, and context-aware spatial applications seamlessly.

By leveraging JSAI, you can integrate artificial intelligence capabilities into your spatial experiences, enhancing how users interact with digital content in a 3D environment.

## Documentation

For more detailed information, tutorials, and API references, please visit our [official documentation](https://jsar-project.github.io/jsai/).
